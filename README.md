# UCMAS Karapitiya Website

This is a Vite + React website package prepared for deployment to Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a GitHub repository and upload these files.
2. Sign in to Vercel.
3. Import the GitHub repository.
4. Framework preset: **Vite**.
5. Build command: `npm run build`
6. Output directory: `dist`
7. Deploy.
